# Stock_monitor
Monitor the stock market in China

Python 3.7+

依赖的包
1. numpy
2. pandas
3. PyQt5
4. pyqtgraph
5. PyQtChart
6. qdarkstyle
7. execjs
8. BeautifulSoup
9. psutil
10. akshare
11. baostock
12. matplotlib

每个包的安装，直接用 pip install package


#如何使用

1.下载Stock_monitor
2.打开terminal，cd到Stock_monitor下
3.运行python Stock_Monitor_GUI.py
